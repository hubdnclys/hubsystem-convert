#### 상품관리 API 서버 개발

- 새로운 회사에서 일하게 된 사원님은 상품관리 API 서버를 개발을 맡았습니다.
- 다행히 전임자가 기본 세팅은 해놓았다고 합니다! 개발 요건을 잘 확인하고, 완성해 주세요.

아래 요구 사항을 해결해주세요.
- FEATURE 1. JWT를 이용한 로그인 / 회원가입 API가 있어야 합니다.
- FEATURE 2. 상품을 읽기, 생성, 저장, 삭제할 수 있는 API가 있어야 합니다.
